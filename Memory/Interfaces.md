# Interfaces

## Будущие интерфейсы памяти

### Входы
- журналы фактов;
- утверждённые документы;
- профили и роли;
- стабилизированные термины.

### Выходы
- краткие устойчивые записи памяти;
- индексы повторяющихся решений;
- подсказки по повторному использованию проверенных паттернов.

## Контракт `0.1.0`
В `0.1.0` интерфейсы памяти только описываются. Реальные средства записи, извлечения и поиска не реализуются.

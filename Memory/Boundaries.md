# Boundaries

## Границы домена
- `Memory/` не подменяет `Docs/`;
- `Memory/` не подменяет `Logs/`;
- `Memory/` не хранит текущее исполнение из `Runtime/`;
- `Memory/` не может вводить новые правила без отражения в `Rules/` и `Standards/`.

## Допустимое происхождение памяти
Будущие записи памяти могут появляться только после анализа уже зафиксированных артефактов:
- `ADRlog.md`
- `ChangeLog.md`
- `ReleaseLog.md`
- `SupportLog.md`
- утверждённых документов `Docs/`

## Статус `v1`
Границы зафиксированы, но сама память не активируется.

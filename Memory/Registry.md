# Registry

## Индекс будущих типов памяти
- `MEM-000001` — профильная память
- `MEM-000002` — продуктовая память
- `MEM-000003` — процессная память

---

## MEM-000001 — профильная память
ID: MEM-000001
Название: профильная память
Статус: Резерв
Связи: PROF-000001, PROF-000002
Источник: Каркас домена `Memory`
Дата_создания: 2026-03-10
Дата_изменения: 2026-03-10

### Описание
В будущем может хранить устойчивые особенности бренда, режима строгости и повторяющихся решений профиля.

# Templates

`Templates/` хранит человеко-ориентированные заготовки документов.

## Правила
- шаблон обязан зеркалить обязательные поля схемы;
- дополнительные разделы допускаются только для повышения читаемости;
- шапка записи не заменяется свободным текстом;
- поле `Связи` содержит только идентификаторы;
- для монолитных файлов запись оформляется отдельным блоком Markdown.

## Текущий контракт
- все примерные `ID` в шаблонах используют 6-значную числовую часть;
- `Profiles/` в `BytePress` используют semantic filename, но внутренний `PROF ID` и связанные ссылки приводятся к 6-значному формату;
- commit message, PR title и PR body оформляются на английском;
- язык пользовательского взаимодействия определяется параметром профиля `Язык_взаимодействия`.

## Шаблоны продукта
- `JTBD.md` — минимальный шаблон Jobs To Be Done.
- `PRD.md` — минимальный шаблон Product Requirements Document.

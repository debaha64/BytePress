# Inputs_Outputs

## Ключевые переходы
- `Обсуждение` → выход: набор тем и неизвестных.
- `Интервью` → выход: уточнённые ответы по слоям.
- `Исследование` → выход: подтверждённые факты и ограничения.
- `Требования` → выход: оформленные требования.
- `Пакет основания` → выход: `Docs/User/`, `Docs/Product/`, `Docs/Technical/`, `Docs/Terms/`.
- `Roadmap` → выход: `Plans/Roadmap.md`.
- `Backlog` → выход: `Plans/Backlog.md`.
- `План` → вход: backlog-item + требования + при необходимости решение из `ADRlog`.
- `Проверка` → выход: записи в `Logs/QualityLog.md`.
- `Реализация` и `Выпуск` → выход: записи в `ChangeLog.md` и `ReleaseLog.md`.

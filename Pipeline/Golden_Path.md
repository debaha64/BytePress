# Golden_Path

## Основной путь BytePress
`Запрос → Стартовый отчёт → Синхронизация develop → Рабочая ветка → ROAD/BACK/PLAN → Документы-владельцы → Противоречия и гипотезы → Изменения → Проверки → Журнальное закрытие → Смысловые коммиты → Push → PR через gh`

## Назначение
Этот путь обязателен для профиля `Default` с максимальной строгостью. Подробные процедуры живут в `Pipeline/Workflows.md`.

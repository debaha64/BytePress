# Terminology

ID: STD-0005
Название: Terminology
Статус: Активно
Связи: RULE-0006, TERM-0001, TERM-0015, TERM-0016
Источник: BytePress + полезные практики Speculorg
Дата_создания: 2026-03-09
Дата_изменения: 2026-03-10

## Назначение
Нормализация терминов и управление конфликтами словаря BytePress и продуктовых словарей.

## Обязательные_требования
- Новый термин нельзя вводить без проверки конфликта с уже существующими терминами.
- Карточка термина содержит ID, определение, статус, синонимы, связанные термины, источник и решение по конфликту.
- Определение термина хранится только в карточке термина.
- Ссылки на термины в системных документах даются по ID.
- Для каждого нового термина принимается одно решение: принять, заменить, запретить.

## Критерии_соблюдения
- Термин оформлен отдельным файлом по шаблону.
- Решение по конфликту заполнено.
- Синонимы и связанные термины указаны явно.
- Термин включён в индекс словаря.

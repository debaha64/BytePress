# Traceability

ID: STD-000008
Название: Traceability
Статус: Активно
Связи: RULE-000003, RULE-000008, RULE-000010
Источник: BytePress + полезные практики Speculorg
Дата_создания: 2026-03-10
Дата_изменения: 2026-04-06

## Назначение
Ключевые сущности должны иметь стабильные идентификаторы и явные связи.

## Обязательные_требования
- Сущности, для которых контракт требует `ID`, не могут оставаться без него в active layer.
- Связи между артефактами фиксируются по каноническим `ID`, а path используется только как навигационное дополнение, когда это нужно.
- Migration `ID` не должна ломать traceability между артефактами, журналами и плановым контуром.

## Критерии_соблюдения
- У активных артефактов нет конфликтующих или двусмысленных идентификаторов.
- Связанные артефакты можно однозначно сопоставить по `ID`.
- Прямые path-references не спорят с каноническими ссылками по `ID`.

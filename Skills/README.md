# Skills

`Skills/` — локальная библиотека проверенных процедурных способностей BytePress.

## Назначение
Навык описывает повторяемую процедуру. Он не хранит системное знание, не заменяет роль и не дублирует инструмент.

## Принципы
- навык соответствует одной фазе или одной устойчивой процедуре;
- навык указывает входы, выходы и шаги;
- навык ссылается на роли, правила, стандарты и инструменты, но не копирует их содержимое;
- навык не зависит от конкретной модели;
- навык должен быть коротким и исполнимым.

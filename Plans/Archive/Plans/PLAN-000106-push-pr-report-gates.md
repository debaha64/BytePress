# PLAN-000106 — Гейты перед push/PR и итоговый отчёт агента

ID: PLAN-000106
Название: Гейты перед push/PR и итоговый отчёт агента
Статус: Завершено
Связи: ROAD-000045, BACK-000118, CHG-000118, QL-000113
Источник: Запрос владельца от 2026-05-22
Дата_создания: 2026-05-22
Дата_изменения: 2026-05-22
Основание: Владелец запросил выполнить `BACK-000118` в рамках `ROAD-000045`: закрепить гейты перед `push`/PR, стандарт краткого итогового отчёта агента и закрыть этап только после завершения `BACK-000116`, `BACK-000117` и `BACK-000118`.
Связанные_требования:
- Перед `push` агент проверяет Git-состояние, diff, плановый контур, связанные артефакты и обязательные проверки.
- Перед PR рабочая ветка отправлена в remote, проверки пройдены, описание PR содержит пакет доказательств.
- PR создаётся через `gh`.
- Итоговый отчёт агента содержит ветку, PR, использованные ID, изменённые артефакты, commit-ы, проверки, состояние `ROAD/BACK/PLAN`, риски, что не трогалось и рекомендацию.
- `ROAD-000045` закрывается только если `BACK-000116`, `BACK-000117` и `BACK-000118` завершены.

- Итоговый отчёт задаёт обязательный минимум, но не заменяет фактическое описание PR и журнальные записи.
Определение_готовности:
- `BACK-000118` и `PLAN-000106` завершены.
- Гейт перед отправкой ветки описывает Git-состояние, diff, плановый контур, связанные артефакты и обязательные проверки.
- Гейт перед PR требует отправленную remote-ветку, пройденные проверки, пакет доказательств в описании PR и создание PR через `gh`.
- Итоговый отчёт агента содержит обязательный краткий набор полей.
- `ROAD-000045` завершён только после подтверждения завершённости `BACK-000116`, `BACK-000117` и `BACK-000118`.
- `ROAD-000046`, `Tools/*`, product bootstrap, проверочная архитектура и zip-архивы не затронуты.
- `git diff --check` и `python3 Tools/bp_lint.py --repo .` выполнены.

## Шаги

   - Описание: Добавить `BACK-000118`, создать `PLAN-000106` и связать их с `ROAD-000045`.
   - Определение_готовности: `Plans/Roadmap.md`, `Plans/Backlog.md`, `Plans/Archive/Backlog/ROAD-000045.md` и текущий план согласованы.
2. Закрепить гейты перед `push` и PR.
   - Описание: Уточнить `Pipeline/Workflows.md` и `Rules/Git.md`.
   - Определение_готовности: Договор описывает минимальные условия перед отправкой ветки, минимальные условия перед PR и маршрут PR через `gh`.
3. Закрепить итоговый отчёт агента.
   - Описание: Уточнить `Pipeline/Workflows.md` и `Rules/Workflow.md`.

   - Описание: Завершить `BACK-000118`, `PLAN-000106` и `ROAD-000045`, добавить `CHG-000118` и `QL-000113`.
   - Определение_готовности: Журналы содержат факты, проверки пройдены, `ROAD-000046` не начат.

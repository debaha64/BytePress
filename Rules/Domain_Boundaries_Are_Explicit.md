# Domain_Boundaries_Are_Explicit

ID: RULE-000007
Название: Domain_Boundaries_Are_Explicit
Статус: Активно
Связи: ADR-000002, ADR-000005
Источник: Базовый регламент BytePress
Дата_создания: 2026-03-10
Дата_изменения: 2026-03-10
Обязательность: Обязательно
Связанные_правила:
- RULE-000003

## Назначение
Каждый домен системы должен хранить только свой класс артефактов и не подменять соседние домены.

## Нарушение
`Docs/` начинает хранить временное исполнение, tool-output path хранит каноническое знание, `Logs/` подменяют планы или `Rules/` дублируют технические договоры.

## Проверка
Проверка размещения артефактов и их соответствия доменной карте системы.

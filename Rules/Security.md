# Security

## RULE-000002 — No secrets in Git
ID: RULE-000002
Название: No secrets in Git
Статус: Активно
Связи: RULE-000003
Источник: Базовый регламент BytePress
Дата_создания: 2026-03-09
Дата_изменения: 2026-04-30
Обязательность: Обязательно

### Назначение
Рабочие секреты, реальные конфигурации и чувствительные данные не хранятся в Git.

### Нарушение
Секреты попадают в репозиторий, в шаблоны, в журналы или в документацию системы.

### Проверка
Проверка конфигурационных файлов, шаблонов окружения и содержимого репозитория.

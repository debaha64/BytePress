# Technical

`Docs/Technical/` содержит долговременное техническое знание о самой системе `BytePress`.

## Состав
- `Architecture.md` — архитектурная карта доменов и их границ.
- `Artifact_Lifecycle.md` — канонический contract жизненного цикла артефактов и порядка их синхронизации.
- `Model.md` — доменная модель сущностей и их отношений.
- `Pipeline.md` — техническая роль конвейера и его привязка к доменам.
- `Interfaces.md` — внутренние и внешние интерфейсы системы.
- `Platform_Contracts.md` — базовые контракты среды выполнения и рабочего периметра.
- `Product_Bootstrap_Contract.md` — технический контракт минимального bootstrap продукта.
- `Product_Bootstrap_Validation.md` — зафиксированный результат проверки bootstrap продукта.
- `System_Invariants.md` — системные инварианты, нарушение которых недопустимо.

## Границы
- этот каталог не хранит оперативное состояние исполнения;
- этот каталог не дублирует правила, стандарты, планы и журналы;
- этот каталог описывает устройство системы, а не ход конкретной задачи.

## Связи
- конвейер как модель процесса раскрыт в `../../Pipeline/`;
- порядок обязательной синхронизации артефактов раскрыт в `Artifact_Lifecycle.md`;
- аналитический current-truth слой раскрыт в `../Discovery/`;
- ограничения и запреты закреплены в `../../Rules/`;
- нормативы качества и оформления закреплены в `../../Standards/`;
- термины системы закреплены в `../Terms/`.

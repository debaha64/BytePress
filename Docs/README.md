# Docs

`Docs/` хранит только долговременное знание.

## Слои
- `User/` — знания для человека-пользователя системы.
- `Discovery/` — аналитическое долговременное знание и current-truth интервью.
- `Product/` — продуктовая рамка и поставка.
- `Technical/` — техническая модель системы и contracts жизненного цикла артефактов.
- `Terms/` — терминологическая база BytePress.

## Правило
Оперативное состояние, временные заметки и текущие рабочие файлы не должны храниться в `Docs/`.

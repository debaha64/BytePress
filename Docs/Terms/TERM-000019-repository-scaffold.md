Связи: TERM-000003, TERM-000011, TERM-000012, TERM-000018

Дата_изменения: 2026-04-28

## Определение
Минимальный управляемый каркас репозитория продукта, который bootstrap materialize по умолчанию для раннего product-start: корневые точки входа, ограниченный `Docs/*`, `Plans/*`, `Logs/*`, лёгкий `Pipeline/*`, локальный `Tools/*`, bounded `Templates/*`, bounded `Schemas/*` и переходный `scripts/*` слой только как оболочки к `Tools/*`.

- Repository Scaffold

- TERM-000012

## Уточнение
Каркас репозитория описывает только bootstrap-default основу product repo.

Он не означает:
- полную копию доменной структуры `BytePress`;
- автоматический перенос `Adapters/`, `Memory/`, `MCP/`, `Runtime/`, `Roles/`, `Skills/` или `Standards/` в продукт;
- перенос полного фабричного `Tools/*`; продукт получает только локальные product tools;
- превращение переходного `scripts/*` слоя в основной служебный слой;
- разрешение на предметное расширение продукта до отдельного product-pass.

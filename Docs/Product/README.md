# Product

## Назначение
Слой `Product/` описывает BytePress как поставляемый продукт и как фабрику поставки других продуктов.

## Что должен обеспечивать BytePress
- управляемый конвейер от обсуждения до поддержки;
- подготовку пакета основания документации до начала разработки;
- формирование `Roadmap` и `Backlog` на основе требований;
- подготовку продукта к передаче другому разработчику;
- работу через профиль, а не через случайный набор флагов.

## Основные документы слоя
- `Implementation_Plan.md` — план сборки BytePress 0.1.0.
- `Profiles.md` — назначение профилей и их влияние на режим.
- `Delivery.md` — модель поставки и передачи.

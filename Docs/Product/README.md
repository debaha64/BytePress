# Product

## Назначение
Слой `Docs/Product/` описывает BytePress как продукт и фиксирует только прикладное представление первой версии.

## Состав слоя
- `JTBD.md` — прикладные задачи пользователя и ожидаемый ценный результат.
- `PRD.md` — продуктовые требования первой версии.
- `Delivery.md` — краткая модель поставки и передачи продукта.

## Границы
- слой не хранит технические контракты bootstrap и результаты их проверки;
- слой не дублирует plan-layer и не хранит отдельные продуктовые сводки вместо канонических планов;
- слой описывает продуктовую форму BytePress, а не внутреннее устройство всей системы.

## Связи
- техническое устройство системы раскрыто в `../Technical/`;
- плановый контур ведётся в `../../Plans/`;
- журналы изменений и проверки ведутся в `../../Logs/`.
